## jQuery plugin

It's as simple as:

    (function($, undefined) {
       $.fn.myButton = function() {
        this.each(function(button) {
          var text = button.text();
          button.click(function() {
            alert(text);
          });
        });

        return this;
       }
    })(jQuery);

And can be used like this:

    $('button').myButton();
